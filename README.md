# AndroidVersions
تطبيق Android يعرض قائمة بإصدارات نظام Android باستخدام RecyclerView.

## المميزات
- قائمة 12 إصدار Android من Donut إلى Oreo
- CardView لكل عنصر مع أيقونة ملونة
- ألوان متبادلة للصفوف الزوجية والفردية
- Snackbar عند الضغط على أي عنصر
- زر Sort للترتيب أبجدياً أو تنازلياً

## التقنيات المستخدمة
- Java
- ViewBinding
- RecyclerView + CardView
- ConstraintLayout
- Material Snackbar
